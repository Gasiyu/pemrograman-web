## Installation 

```bash
$ git clone https://github.com/Gasiyu/pemrograman-web.git -b praktikum7
$ cd pemrograman-web
$ cp .env.example .env
$ composer install
$ npm install
$ php artisan key:generate
$ php artisan migrate:fresh
```
